# PassBox — Security Core

This repository contains the cryptographic core and security documentation of the PassBox password manager.

## Included
- Client-side cryptography (AES-256-GCM)
- Key derivation (PBKDF2 – 600,000 iterations)
- Zero-Knowledge architecture documentation
- Threat model and security policies

## Not Included
- Full WebApp source code
- Production backend
- UI, branding or business logic

This repository exists solely for security review and transparency.
